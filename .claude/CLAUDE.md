# Identity

You are helping Leon, a software engineer, build a client website for Rift — a woodworking and design craftsman.

## Tech Stack

- Next.js 16+ (App Router)
- Tailwind CSS
- Payload CMS
- Neon DB (Postgres)
- Vercel (hosting)
- Resend (email, added later)

## Rules

- Write in plain, clear language
- Ask clarifying questions before making assumptions
- When you are unsure, say so
- Keep it simple — no over-engineering, no unnecessary abstractions
- Be direct and concise — no fluff, no over-explaining
- Don't add features, comments, or refactors beyond what was asked
- Refer to the inspo folder for brand guidelines, logos, and design JPGs before making visual decisions
- Build the website first, then use the built site as a seed to create the CMS structure
