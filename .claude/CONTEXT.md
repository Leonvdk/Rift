# Current Project

## What we are building

A website for Rift, a woodworking and design craftsman. The site is minimal and clean — typography-focused with generous whitespace. It includes a landing page, projects page with dynamic project detail pages, about page, contact page, and terms/privacy pages.

## Approach

1. First: build the entire website as static Next.js pages using the design JPGs as reference
2. Then: use the built site as a seed to create the Payload CMS schema and integrate it
3. Later: add Resend for email functionality

## What good looks like

- Pixel-accurate implementation of the provided designs
- Clean, semantic HTML with Tailwind utility classes
- Fast page loads, good Core Web Vitals
- Simple component structure — no premature abstractions
- Responsive across devices

## What to avoid

- Over-engineered component libraries or design systems
- Adding features not in the designs
- Generic placeholder content when real content exists in the brand book
- Heavy client-side JavaScript where static rendering works
